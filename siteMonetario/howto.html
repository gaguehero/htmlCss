Guia de Uso

Para fazer uma busca no TrocaGrana é muito fácil. Para escolher qual tipo de moeda, assinale a opção correspondente. O usuário pode escolher entre três moedas disponíveis: dólar americano, euro e libra. É necessário selecionar ao menos uma moeda para fazer a busca. O usuário pode escolher mais de 1 moeda ao mesmo tempo.

Para selecionar um tempo de busca, temos dois campos: Data inicial e Data final. Selecione a data final como equivalente ao dia inicial da pesquisa, e data final como equivalente ao dia final. Caso o usuário selecione os valores invertidos, o TrocaGrana emitirá um alerta, mas efetuará a busca ajustando os campos automaticamente.

Após a busca, duas informações serão exibidas: um gráfico de linhas e uma tabela com os valores, contendo a cotação e a data/hora em questão. Caso o usuário queira fazer uma nova busca, os dados serão atualizados automaticamente.

O TrocaGrana ainda é um protótipo, então certas falhas e bugs são esperadas, caso ocorra algum problema, reporte para contato@trocagrana.com.br. Agradecemos pela ajuda.
